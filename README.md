# FocusFlow ⏱️

FocusFlow is a modern Pomodoro productivity app that helps you stay focused with music, smart scheduling, and study tracking.

---

## ✨ Features

- ⏱ Custom Pomodoro Timer (focus + break)
- 🎧 Music search (type and play using YouTube)
- 📊 Subject-wise study tracking
- 📅 User-defined study schedule
- 🎨 Clean and minimal UI

---

## 🛠 Tech Stack

- React (Vite)
- TypeScript
- CSS
- YouTube Data API
